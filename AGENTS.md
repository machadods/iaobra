# INSTRUÇÕES DO PROJETO — IAObras / Simulador Temporal ## Quem sou eu Meu nome é Wagner. Me refiro a mim mesmo como Lorde. Sou desenvolvedor amador com boa capacidade de adaptação. Uso VS Code, PowerShell no Windows, Python com venv, Streamlit para interfaces, PostgreSQL + DBeaver para banco de dados. ## O que é este projeto IAObras é uma plataforma de acompanhamento de obras com IA. A visão central é criar um **Simulador de Estado Temporal da Obra**: gravando em dados, imagens e vídeos tudo que ocorre na obra, criar uma experiência onde no futuro alguém pode "viajar no tempo" e ver como o carpinteiro fez a caixaria da fundação — como se fosse ao vivo, em VR/AR. É uma combinação de diário de obra com IA + arquivo vivo + simulação temporal. ## Stack Técnica - Python 3.x - Streamlit (interface) - PostgreSQL + DBeaver (banco de dados) - Integração futura: VR/AR, upload de imagens/vídeos, LLM para análise ## Estado Atual — Esqueleto Inicial (nov/2025) ### O que existe: - app.py — entrada principal - iaobras.py — lógica principal - orcamento.py — módulo de orçamento - sobras.py — controle de sobras/materiais - tela_diario.py — diário da obra - tela_obras.py — listagem de obras - tela_estudo_mestre.py — tela mais desenvolvida (5kb) - style.py — estilos ### Estrutura:
