## 🚀 PRÓXIMOS PASSOS (30-60 minutos)

### **Passo 1: Instalar** (5 min)
```powershell
pip install -r requirements.txt
```

### **Passo 2: Criar BD** (10 min)
```powershell
psql -U postgres -d iaobras_db -f database/migrations/v003_media_stats_schema.sql
```

### **Passo 3: Integrar** (5 min)
Adicionar em `app.py`:
- Menu: `"analise": ("📊 Análise de Dados", "analise")`
- Render: `elif pagina == "analise": render()`

### **Passo 4: Testar** (5 min)
```powershell
streamlit run app.py
```
